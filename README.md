# Car Dealership Console Application

## Description of the Project
This application is a dealership system that allows users to search, view, add, and remove vehicles from a dealership inventory.

The application reads dealership and vehicle data from an `inventory.csv` file, allows users to search vehicles using different filters. Then Changes inventory based on user input.

## User Stories
- As a user, I want to view all vehicles, so that I can see the dealership inventory.
- As a user, I want to search vehicles by price range, so that I can find vehicles within my budget.
- As a user, I want to search vehicles by make and model, so that I can quickly find a specific vehicle.
- As a user, I want to search vehicles by year range, so that I can find newer or older vehicles.
- As a user, I want to search vehicles by color, so that I can find vehicles I like.
- As a user, I want to search vehicles by mileage range, so that I can find vehicles with lower mileage.
- As a user, I want to search vehicles by type, so that I can filter cars, trucks, SUVs, and vans.
- As a user, I want to add vehicles, so that I can update dealership inventory.
- As a user, I want to remove vehicles, so that sold vehicles are no longer listed.

## Setup

### Prerequisites
- IntelliJ IDEA
- Java SDK
- `inventory.csv` file

### Running the Application in IntelliJ
1. Open IntelliJ IDEA.
2. Select **Open** and navigate to the project folder.
3. Wait for IntelliJ to index the files and set up the project.
4. Locate the `Program` class.
5. Make sure `inventory.csv` is in the correct project directory.
6. Right-click `Program.java`.
7. Select **Run 'Program.main()'**.
